# Claude guidance

This is a Rust library/binary with optional Python bindings via PyO3/maturin. CI is
split into per-job workflow files under `.github/workflows/`. Pre-commit hooks are
enforced locally via `.pre-commit-config.yaml`. `cargo-deny` enforces licence and
advisory policy via `deny.toml`.
